\doxysubsection{Detailed Description}
\textquotesingle{}\mbox{\hyperlink{class_fl___preferences_1_1_name}{Name}}\textquotesingle{} provides a simple method to create numerical or more complex procedural names for entries and groups on the fly.

Example\+: prefs.\+set(\mbox{\hyperlink{class_fl___preferences_1_1_name}{Fl\+\_\+\+Preferences\+::\+Name}}(\char`\"{}\+File\%d\char`\"{},i),file\mbox{[}i\mbox{]});.

See test/preferences.\+cxx as a sample for writing arrays into preferences.

\textquotesingle{}\mbox{\hyperlink{class_fl___preferences_1_1_name}{Name}}\textquotesingle{} is actually implemented as a class inside \mbox{\hyperlink{class_fl___preferences}{Fl\+\_\+\+Preferences}}. It casts into const char$\ast$ and gets automatically destroyed after the enclosing call ends.
