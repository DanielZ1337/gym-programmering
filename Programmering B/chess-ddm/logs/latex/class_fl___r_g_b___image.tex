\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} class supports caching and drawing of full-\/color images with 1 to 4 channels of color information. Images with an even number of channels are assumed to contain alpha information, which is used to blend the image with the contents of the screen.

\mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} is defined in $<$\mbox{\hyperlink{_fl___image_8_h}{F\+L/\+Fl\+\_\+\+Image.\+H}}$>$, however for compatibility reasons $<$\mbox{\hyperlink{_fl___r_g_b___image_8_h}{F\+L/\+Fl\+\_\+\+R\+G\+B\+\_\+\+Image.\+H}}$>$ should be included.
