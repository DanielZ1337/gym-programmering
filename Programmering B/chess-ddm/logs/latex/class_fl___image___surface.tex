\doxysubsection{Detailed Description}
Directs all graphics requests to an \mbox{\hyperlink{class_fl___image}{Fl\+\_\+\+Image}}.

After creation of an \mbox{\hyperlink{class_fl___image___surface}{Fl\+\_\+\+Image\+\_\+\+Surface}} object, call \mbox{\hyperlink{class_fl___image___surface_a4b8a38a196aa91bfded34751792b237b}{set\+\_\+current()}} on it, and all subsequent graphics requests will be recorded in the image. It\textquotesingle{}s possible to draw widgets (using \mbox{\hyperlink{class_fl___image___surface_a0c4a97f89c9b0fc091d8d2c0be7da0ce}{Fl\+\_\+\+Image\+\_\+\+Surface\+::draw()}}) or to use any of the \mbox{\hyperlink{group__fl__drawings}{Drawing functions}} or the \mbox{\hyperlink{group__fl__attributes}{Color \& Font functions}}. Finally, call \mbox{\hyperlink{class_fl___image___surface_ab9034b7cc16d89886896b82b72b7e29b}{image()}} on the object to obtain a newly allocated \mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} object. ~\newline
 Fl\+\_\+\+G\+L\+\_\+\+Window objects can be drawn in the image as well.
