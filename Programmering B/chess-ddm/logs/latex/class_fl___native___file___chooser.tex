\doxysubsection{Detailed Description}
This class lets an F\+L\+TK application easily and consistently access the operating system\textquotesingle{}s native file chooser. Some operating systems have very complex and specific file choosers that many users want access to specifically, instead of F\+L\+TK\textquotesingle{}s default file chooser(s).

In cases where there is no native file browser, F\+L\+TK\textquotesingle{}s own file browser is used instead.

To use this widget, use the following include in your code\+:

\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{_fl___native___file___chooser_8_h}{FL/Fl\_Native\_File\_Chooser.H}}>}}
\end{DoxyCode}

The \mbox{\hyperlink{class_fl___native___file___chooser}{Fl\+\_\+\+Native\+\_\+\+File\+\_\+\+Chooser}} widget transmits U\+T\+F-\/8 encoded filenames to its user. It is recommended to open files that may have non-\/\+A\+S\+C\+II names with the \mbox{\hyperlink{group__fl__unicode_gab798a9fb10c29ebe56f292b4cc8e8022}{fl\+\_\+fopen()}} or \mbox{\hyperlink{group__fl__unicode_ga2e60bfbced4b10ac703ad6817b59398b}{fl\+\_\+open()}} utility functions that handle these names in a cross-\/platform way (whereas the standard fopen()/open() functions fail on the M\+S\+Windows platform to open files with a non-\/\+A\+S\+C\+II name).

{\bfseries{Platform Specific Caveats}}


\begin{DoxyItemize}
\item Under X windows, and if \mbox{\hyperlink{class_fl_a43e6e0bbbc03cad134d928d4edd48d1da01d530d6bc747eb0c192038eb802a8bd}{Fl\+::\+O\+P\+T\+I\+O\+N\+\_\+\+F\+N\+F\+C\+\_\+\+U\+S\+E\+S\+\_\+\+G\+TK}} has not been switched off, the widget attempts to use standard G\+TK file chooser dialogs if they are available at run-\/time on the platform, and falls back to use F\+L\+TK\textquotesingle{}s \mbox{\hyperlink{class_fl___file___chooser}{Fl\+\_\+\+File\+\_\+\+Chooser}} if they are not. In the latter case, it\textquotesingle{}s best if you call \mbox{\hyperlink{class_fl___file___icon_aca0613c70b701060f3335c1ef5af7ef0}{Fl\+\_\+\+File\+\_\+\+Icon\+::load\+\_\+system\+\_\+icons()}} at the start of \mbox{\hyperlink{main_8cpp_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()}}, to enable the nicer looking file browser widgets. Use the static public attributes of class \mbox{\hyperlink{class_fl___file___chooser}{Fl\+\_\+\+File\+\_\+\+Chooser}} to localize the browser.
\item Some operating systems support certain OS specific options; see \mbox{\hyperlink{class_fl___native___file___chooser_a9c0256b7db881087e8d9bdd953ddf60a}{Fl\+\_\+\+Native\+\_\+\+File\+\_\+\+Chooser\+::options()}} for a list.
\end{DoxyItemize}
