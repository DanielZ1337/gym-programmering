\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___color___chooser}{Fl\+\_\+\+Color\+\_\+\+Chooser}} widget provides a standard R\+GB color chooser. 

 
\begin{DoxyImage}
\includegraphics[width=5cm]{fl_color_chooser.jpg}
\doxyfigcaption{fl\+\_\+color\+\_\+chooser()}
\end{DoxyImage}


You can place any number of the widgets into a panel of your own design. The diagram shows the widget as part of a color chooser dialog created by the \mbox{\hyperlink{_fl___color___chooser_8_h_a12b867658f689ffcd6848cda456799a7}{fl\+\_\+color\+\_\+chooser()}} function. The \mbox{\hyperlink{class_fl___color___chooser}{Fl\+\_\+\+Color\+\_\+\+Chooser}} widget contains the hue box, value slider, and rgb input fields from the above diagram (it does not have the color chips or the Cancel or OK buttons). The callback is done every time the user changes the rgb value. It is not done if they move the hue control in a way that produces the {\itshape same} rgb value, such as when saturation or value is zero.

The \mbox{\hyperlink{_fl___color___chooser_8_h_a12b867658f689ffcd6848cda456799a7}{fl\+\_\+color\+\_\+chooser()}} function pops up a window to let the user pick an arbitrary R\+GB color. They can pick the hue and saturation in the \char`\"{}hue box\char`\"{} on the left (hold down C\+T\+RL to just change the saturation), and the brightness using the vertical slider. Or they can type the 8-\/bit numbers into the R\+GB \mbox{\hyperlink{class_fl___value___input}{Fl\+\_\+\+Value\+\_\+\+Input}} fields, or drag the mouse across them to adjust them. The pull-\/down menu lets the user set the input fields to show R\+GB, H\+SV, or 8-\/bit R\+GB (0 to 255).

\mbox{\hyperlink{_fl___color___chooser_8_h_a12b867658f689ffcd6848cda456799a7}{fl\+\_\+color\+\_\+chooser()}} returns non-\/zero if the user picks ok, and updates the R\+GB values. If the user picks cancel or closes the window this returns zero and leaves R\+GB unchanged.

If you use the color chooser on an 8-\/bit screen, it will allocate all the available colors, leaving you no space to exactly represent the color the user picks! You can however use \mbox{\hyperlink{group__fl__drawings_ga2986a868e9cc8d9141acde94c0fe8ab0}{fl\+\_\+rectf()}} to fill a region with a simulated color using dithering.
