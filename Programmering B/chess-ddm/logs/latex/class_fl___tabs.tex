\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}} widget is the \char`\"{}file card tabs\char`\"{} interface that allows you to put lots and lots of buttons and switches in a panel, as popularized by many toolkits.

 
\begin{DoxyImage}
\includegraphics[width=8cm]{tabs.png}
\doxyfigcaption{Fl\+\_\+\+Tabs}
\end{DoxyImage}


Clicking the tab makes a child \mbox{\hyperlink{class_fl___widget_ac103c73444da9035a31329cf938d77b3}{visible()}} by calling \mbox{\hyperlink{class_fl___widget_a3b7971dc3a0385c31e71b9f3786eac2d}{show()}} on it, and all other children are made invisible by calling \mbox{\hyperlink{class_fl___widget_a408b7ca0d0ed7682275e79b976c5dda4}{hide()}} on them. Usually the children are \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}} widgets containing several widgets themselves.

Each child makes a card, and its \mbox{\hyperlink{class_fl___widget_aa327c70689110677d96067da55e227aa}{label()}} is printed on the card tab, including the label font and style. The selection color of that child is used to color the tab, while the color of the child determines the background color of the pane.

The size of the tabs is controlled by the bounding box of the children (there should be some space between the children and the edge of the \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}}), and the tabs may be placed \char`\"{}inverted\char`\"{} on the bottom -\/ this is determined by which gap is larger. It is easiest to lay this out in fluid, using the fluid browser to select each child group and resize them until the tabs look the way you want them to.

The background area behind and to the right of the tabs is \char`\"{}transparent\char`\"{}, exposing the background detail of the parent. The value of \mbox{\hyperlink{class_fl___widget_a9d1f48fbbe4d885e81df38b2a93b614f}{Fl\+\_\+\+Tabs\+::box()}} does not affect this area. So if \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}} is resized by itself without the parent, force the appropriate parent (visible behind the tabs) to \mbox{\hyperlink{class_fl___widget_aa63ce68cbf4620cf8750b868368ea02b}{redraw()}} to prevent artifacts.

See \char`\"{}\+Resizing Caveats\char`\"{} below on how to keep tab heights constant. See \char`\"{}\+Callback\textquotesingle{}s Use Of when()\char`\"{} on how to control the details of how clicks invoke the \mbox{\hyperlink{class_fl___widget_ac32acdcfb3b722fccd2d3f946b01e799}{callback()}}.

{\bfseries{Default}} {\bfseries{Appearance}} 

The appearance of each \char`\"{}tab\char`\"{} is taken from the \mbox{\hyperlink{class_fl___widget_aa327c70689110677d96067da55e227aa}{label()}} and \mbox{\hyperlink{class_fl___widget_a0f4654d4d9aab8a1c1078fca5660742a}{color()}} of the child group corresponding to that \char`\"{}tab\char`\"{} and panel. Where the \char`\"{}tabs\char`\"{} appear depends on the position and size of the child groups that make up the panels within the Fl\+\_\+\+Tab, i.\+e. whether there is more space above or below them. The height of the \char`\"{}tabs\char`\"{} depends on how much free space is available.

\begin{DoxyParagraph}{Callback\textquotesingle{}s Use Of when()}

\end{DoxyParagraph}
As of F\+L\+TK 1.\+3.\+3, \mbox{\hyperlink{class_fl___tabs_ad1a81464c51e3cb56fa1d8fd6ce32d72}{Fl\+\_\+\+Tabs()}} supports the following flags for \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{when()}}\+:


\begin{DoxyItemize}
\item \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665adfa635a2abba2b9d9fbfc10294d87f3c}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+N\+E\+V\+ER}} -- callback never invoked (all flags off)
\item \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ae504aaa4de36a9ff606530956c68887a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+C\+H\+A\+N\+G\+ED}} -- if flag set, invokes callback when a tab has been changed (on click or keyboard navigation)
\item \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ac5385d8670aedf159e0271731fceab85}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+N\+O\+T\+\_\+\+C\+H\+A\+N\+G\+ED}} -- if flag set, invokes callback when the tabs remain unchanged (on click or keyboard navigation)
\item \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665afbd4ae9061c3cd4bc9b4aa94fcb8137a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE}} -- if flag set, invokes callback on R\+E\+L\+E\+A\+SE of mouse button or keyboard navigation
\end{DoxyItemize}

Notes\+:


\begin{DoxyEnumerate}
\item The above flags can be logically O\+R-\/ed ($\vert$) or added (+) to combine behaviors.
\item The default value for \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{when()}} is \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665afbd4ae9061c3cd4bc9b4aa94fcb8137a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE}} (inherited from \mbox{\hyperlink{class_fl___widget}{Fl\+\_\+\+Widget}}).
\item If \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665afbd4ae9061c3cd4bc9b4aa94fcb8137a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE}} is the {\itshape only} flag specified, the behavior will be as if (\mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665afbd4ae9061c3cd4bc9b4aa94fcb8137a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE}}$\vert$\mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ae504aaa4de36a9ff606530956c68887a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+C\+H\+A\+N\+G\+ED}}) was specified.
\item The value of \mbox{\hyperlink{class_fl___widget_af007a3c83ac176a7511df8df0706947c}{changed()}} will be valid during the callback.
\item If both \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ae504aaa4de36a9ff606530956c68887a}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+C\+H\+A\+N\+G\+ED}} and \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ac5385d8670aedf159e0271731fceab85}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+N\+O\+T\+\_\+\+C\+H\+A\+N\+G\+ED}} are specified, the callback is invoked whether the tab has been changed or not. The \mbox{\hyperlink{class_fl___widget_af007a3c83ac176a7511df8df0706947c}{changed()}} method can be used to determine the cause.
\item \mbox{\hyperlink{_enumerations_8_h_adaaa8b52b78abf3ccce4787b112e6665ac5385d8670aedf159e0271731fceab85}{F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+N\+O\+T\+\_\+\+C\+H\+A\+N\+G\+ED}} can happen if someone clicks on an already selected tab, or if a keyboard navigation attempt results in no change to the tabs, such as using the arrow keys while at the left or right end of the tabs. 
\end{DoxyEnumerate}

{\footnotesize\ttfamily int Fl\+\_\+\+Tabs\+::handle (\begin{DoxyParamCaption}\item[{int}]{event }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [virtual]}}

Handles the specified event. You normally don\textquotesingle{}t call this method directly, but instead let F\+L\+TK do it when the user interacts with the widget.

When implemented in a widget, this function must return 0 if the widget does not use the event or 1 otherwise.

Most of the time, you want to call the inherited \mbox{\hyperlink{class_fl___tabs_a73dfe2ab8dbca96e074fcc5af1110e03}{handle()}} method in your overridden method so that you don\textquotesingle{}t short-\/circuit events that you don\textquotesingle{}t handle. In this last case you should return the callee retval.

Returns the tab group for the tab the user has currently down-\/clicked on and remains over until F\+L\+\_\+\+R\+E\+L\+E\+A\+SE. Otherwise, returns N\+U\+LL.
