\doxysubsection{Detailed Description}
This class supports tiling of images over a specified area.

The source (tile) image is {\bfseries{not}} copied unless you call the \mbox{\hyperlink{class_fl___tiled___image_a563becbd904af7e4ce33bd591704b2d9}{color\+\_\+average()}}, \mbox{\hyperlink{class_fl___tiled___image_aaaccf2554a2dcaee2f32945a6183621c}{desaturate()}}, or \mbox{\hyperlink{class_fl___image_aeaf2391ad1c94e246877188851861f5e}{inactive()}} methods.

Draws the image with a bounding box. Arguments {\ttfamily X,Y,W,H} specify a bounding box for the image, with the origin ~\newline
 (upper-\/left corner) of the image offset by the {\ttfamily cx} and {\ttfamily cy} arguments.

In other words\+: {\ttfamily fl\+\_\+push\+\_\+clip(\+X,\+Y,\+W,\+H)} is applied, the image is drawn with its upper-\/left corner at {\ttfamily X-\/cx,Y-\/cy} and its own width and height, {\ttfamily fl\+\_\+pop\+\_\+clip}{\ttfamily ()} is applied.
