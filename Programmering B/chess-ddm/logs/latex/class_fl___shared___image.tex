\doxysubsection{Detailed Description}
This class supports caching, loading, scaling, and drawing of image files.

Most applications will also want to link against the fltk\+\_\+images library and call the \mbox{\hyperlink{_fl___shared___image_8_h_a5c361cb2fdac6c22f6259f5e044657f4}{fl\+\_\+register\+\_\+images()}} function to support standard image formats such as B\+MP, G\+IF, J\+P\+EG, and P\+NG.

Images can be requested (loaded) with \mbox{\hyperlink{class_fl___shared___image_ad80da92bd9878ef2d9a43f4a5139b090}{Fl\+\_\+\+Shared\+\_\+\+Image\+::get()}}, \mbox{\hyperlink{class_fl___shared___image_a6f99c91070e5a63a69daecbafcd4f043}{find()}}, and some other methods. All images are cached in an internal list of shared images and should be released when they are no longer needed. A refcount is used to determine if a released image is to be destroyed with delete.

Draws the image with a bounding box. Arguments {\ttfamily X,Y,W,H} specify a bounding box for the image, with the origin ~\newline
 (upper-\/left corner) of the image offset by the {\ttfamily cx} and {\ttfamily cy} arguments.
