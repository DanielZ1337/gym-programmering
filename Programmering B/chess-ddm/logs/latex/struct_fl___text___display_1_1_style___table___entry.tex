\doxysubsection{Detailed Description}
This structure associates the color, font, and font size of a string to draw with an attribute mask matching attr.

There must be one entry for each style that can be used in an \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}} for displaying text. The style table is an array of struct \mbox{\hyperlink{struct_fl___text___display_1_1_style___table___entry}{Style\+\_\+\+Table\+\_\+\+Entry}}.

The style table is associated with an \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}} by using \mbox{\hyperlink{class_fl___text___display_ae09d61739b737a32868ffe0295a25dec}{Fl\+\_\+\+Text\+\_\+\+Display\+::highlight\+\_\+data()}}.
