\doxysubsection{Detailed Description}
This widget provides a round analog clock display. 

\mbox{\hyperlink{class_fl___clock}{Fl\+\_\+\+Clock}} is provided for Forms compatibility. It installs a 1-\/second timeout callback using \mbox{\hyperlink{class_fl_a6a7f66c2bb04c04cb5252c0466f87022}{Fl\+::add\+\_\+timeout()}}. You can choose the rounded or square type of the clock with \mbox{\hyperlink{class_fl___widget_adda2e8f162b3ea0332eb8d762350bbd5}{type()}}, see below.  ~\newline
