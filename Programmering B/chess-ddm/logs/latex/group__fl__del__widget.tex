\doxysubsection{Detailed Description}
These functions, declared in $<$\mbox{\hyperlink{_fl_8_h}{F\+L/\+Fl.\+H}}$>$, support deletion of widgets inside callbacks. 

\mbox{\hyperlink{group__fl__del__widget_ga5908e37029f697e8b2f86a2f48d98b18}{Fl\+::delete\+\_\+widget()}} should be called when deleting widgets or complete widget trees (\mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}}, \mbox{\hyperlink{class_fl___window}{Fl\+\_\+\+Window}}, ...) inside callbacks.

The other functions are intended for internal use. The preferred way to use them is by using the helper class \mbox{\hyperlink{class_fl___widget___tracker}{Fl\+\_\+\+Widget\+\_\+\+Tracker}}.
