\doxysubsection{Detailed Description}
This input field displays \textquotesingle{}\textbackslash{}n\textquotesingle{} characters as new lines rather than $^\wedge$J, and accepts the Return, Tab, and up and down arrow keys. This is for editing multiline text.

This is far from the nirvana of text editors, and is probably only good for small bits of text, 10 lines at most. Note that this widget does not support scrollbars or per-\/character color control.

If you are presenting large amounts of text and need scrollbars or full color control of characters, you probably want \mbox{\hyperlink{class_fl___text___editor}{Fl\+\_\+\+Text\+\_\+\+Editor}} instead.

In F\+L\+TK 1.\+3.\+x, the default behavior of the \textquotesingle{}Tab\textquotesingle{} key was changed to support consistent focus navigation. To get the older F\+L\+TK 1.\+1.\+x behavior, set \mbox{\hyperlink{class_fl___input___ae23c19c0081ab427fc5ad53d4f503f8d}{Fl\+\_\+\+Input\+\_\+\+::tab\+\_\+nav()}} to 0. Newer programs should consider using \mbox{\hyperlink{class_fl___text___editor}{Fl\+\_\+\+Text\+\_\+\+Editor}}.
