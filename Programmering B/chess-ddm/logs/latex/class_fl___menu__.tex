\doxysubsection{Detailed Description}
Base class of all widgets that have a menu in F\+L\+TK.

Currently F\+L\+TK provides you with \mbox{\hyperlink{class_fl___menu___button}{Fl\+\_\+\+Menu\+\_\+\+Button}}, \mbox{\hyperlink{class_fl___menu___bar}{Fl\+\_\+\+Menu\+\_\+\+Bar}}, and \mbox{\hyperlink{class_fl___choice}{Fl\+\_\+\+Choice}}.

The class contains a pointer to an array of structures of type \mbox{\hyperlink{struct_fl___menu___item}{Fl\+\_\+\+Menu\+\_\+\+Item}}. The array may either be supplied directly by the user program, or it may be \char`\"{}private\char`\"{}\+: a dynamically allocated array managed by the \mbox{\hyperlink{class_fl___menu__}{Fl\+\_\+\+Menu\+\_\+}}.

When the user clicks a menu item, \mbox{\hyperlink{class_fl___menu___a760812b12dc473781b132e6a0a7c33e3}{value()}} is set to that item and then\+:


\begin{DoxyItemize}
\item If the \mbox{\hyperlink{struct_fl___menu___item}{Fl\+\_\+\+Menu\+\_\+\+Item}} has a callback set, that callback is invoked with any userdata configured for it. (The \mbox{\hyperlink{class_fl___menu__}{Fl\+\_\+\+Menu\+\_\+}} widget\textquotesingle{}s callback is N\+OT invoked.)
\item For any Fl\+\_\+\+Menu\+\_\+\+Items that {\bfseries{don\textquotesingle{}t}} have a callback set, the \mbox{\hyperlink{class_fl___menu__}{Fl\+\_\+\+Menu\+\_\+}} widget\textquotesingle{}s callback is invoked with any userdata configured for it. The callback can determine which item was picked using \mbox{\hyperlink{class_fl___menu___a760812b12dc473781b132e6a0a7c33e3}{value()}}, \mbox{\hyperlink{class_fl___menu___a08e0e360e4058971fc3b674dcfdf6180}{mvalue()}}, \mbox{\hyperlink{class_fl___menu___abe4ba98d1af637c1e452df648c19502c}{item\+\_\+pathname()}}, etc. 
\end{DoxyItemize}
