\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_fl___widget}{Fl\+\_\+\+Widget}} is the base class for all widgets in F\+L\+TK. ~\newline


You can\textquotesingle{}t create one of these because the constructor is not public. However you can subclass it. ~\newline


All \char`\"{}property\char`\"{} accessing methods, such as \mbox{\hyperlink{class_fl___widget_a0f4654d4d9aab8a1c1078fca5660742a}{color()}}, \mbox{\hyperlink{class_fl___widget_a6f55261683ecaf68d70e424213d030b0}{parent()}}, or \mbox{\hyperlink{class_fl___widget_a217b954bb7a3c053850de3234fce3e62}{argument()}} are implemented as trivial inline functions and thus are as fast and small as accessing fields in a structure. Unless otherwise noted, the property setting methods such as color(n) or label(s) are also trivial inline functions, even if they change the widget\textquotesingle{}s appearance. It is up to the user code to call \mbox{\hyperlink{class_fl___widget_aa63ce68cbf4620cf8750b868368ea02b}{redraw()}} after these.

Makes a widget visible.

An invisible widget never gets redrawn and does not get keyboard or mouse events, but can receive a few other events like F\+L\+\_\+\+S\+H\+OW.

The \mbox{\hyperlink{class_fl___widget_ac103c73444da9035a31329cf938d77b3}{visible()}} method returns true if the widget is set to be visible. The \mbox{\hyperlink{class_fl___widget_ac51a7380e249127ca47aad175f162360}{visible\+\_\+r()}} method returns true if the widget and all of its parents are visible. A widget is only visible if \mbox{\hyperlink{class_fl___widget_ac103c73444da9035a31329cf938d77b3}{visible()}} is true on it {\itshape and all of its parents}.

Changing it will send F\+L\+\_\+\+S\+H\+OW or F\+L\+\_\+\+H\+I\+DE events to the widget. {\itshape Do not change it if the parent is not visible, as this will send false F\+L\+\_\+\+S\+H\+OW or F\+L\+\_\+\+H\+I\+DE events to the widget}. \mbox{\hyperlink{class_fl___widget_aa63ce68cbf4620cf8750b868368ea02b}{redraw()}} is called if necessary on this or the parent.

Sets the flags used to decide when a callback is called.

This controls when callbacks are done. The following values are useful, the default value is F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE\+:

\begin{DoxyItemize}
\item 0\+: The callback is not done, but \mbox{\hyperlink{class_fl___widget_af007a3c83ac176a7511df8df0706947c}{changed()}} is turned on. \item F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+C\+H\+A\+N\+G\+ED\+: The callback is done each time the text is changed by the user. \item F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+R\+E\+L\+E\+A\+SE\+: The callback will be done when this widget loses the focus, including when the window is unmapped. This is a useful value for text fields in a panel where doing the callback on every change is wasteful. However the callback will also happen if the mouse is moved out of the window, which means it should not do anything visible (like pop up an error message). You might do better setting this to zero, and scanning all the items for \mbox{\hyperlink{class_fl___widget_af007a3c83ac176a7511df8df0706947c}{changed()}} when the OK button on a panel is pressed. \item F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+E\+N\+T\+E\+R\+\_\+\+K\+EY\+: If the user types the Enter key, the entire text is selected, and the callback is done if the text has changed. Normally the Enter key will navigate to the next field (or insert a newline for a \mbox{\hyperlink{class_fl___multiline___input}{Fl\+\_\+\+Multiline\+\_\+\+Input}}) -\/ this changes the behavior. \item F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+E\+N\+T\+E\+R\+\_\+\+K\+E\+Y$\vert$\+F\+L\+\_\+\+W\+H\+E\+N\+\_\+\+N\+O\+T\+\_\+\+C\+H\+A\+N\+G\+ED\+: The Enter key will do the callback even if the text has not changed. Useful for command fields. \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{Fl\+\_\+\+Widget\+::when()}} is a set of bitflags used by subclasses of \mbox{\hyperlink{class_fl___widget}{Fl\+\_\+\+Widget}} to decide when to do the callback.\end{DoxyItemize}
If the value is zero then the callback is never done. Other values are described in the individual widgets. This field is in the base class so that you can scan a panel and \mbox{\hyperlink{class_fl___widget_a37aa870315e37ddc1246caa6421322f4}{do\+\_\+callback()}} on all the ones that don\textquotesingle{}t do their own callbacks in response to an \char`\"{}\+O\+K\char`\"{} button.
