\doxysubsection{Detailed Description}
This widget is a subclass of \mbox{\hyperlink{class_fl___output}{Fl\+\_\+\+Output}} that displays multiple lines of text. It also displays tab characters as whitespace to the next column.

Note that this widget does not support scrollbars, or per-\/character color control.

If you are presenting large amounts of read-\/only text and need scrollbars, or full color control of characters, then use \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}}. If you want to display H\+T\+ML text, use \mbox{\hyperlink{class_fl___help___view}{Fl\+\_\+\+Help\+\_\+\+View}}.
