\doxysubsection{Detailed Description}
Base class for image caching and drawing. 

\mbox{\hyperlink{class_fl___image}{Fl\+\_\+\+Image}} is the base class used for caching and drawing all kinds of images in F\+L\+TK. This class keeps track of common image data such as the pixels, colormap, width, height, and depth. Virtual methods are used to provide type-\/specific image handling.

Since the \mbox{\hyperlink{class_fl___image}{Fl\+\_\+\+Image}} class does not support image drawing by itself, calling the \mbox{\hyperlink{class_fl___image_a95e713f20628bfb4c8155ab94ba65ba1}{draw()}} method results in a box with an X in it being drawn instead.

Draws the image with a bounding box. Arguments {\ttfamily X,Y,W,H} specify a bounding box for the image, with the origin ~\newline
