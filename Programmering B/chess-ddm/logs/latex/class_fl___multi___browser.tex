\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___multi___browser}{Fl\+\_\+\+Multi\+\_\+\+Browser}} class is a subclass of \mbox{\hyperlink{class_fl___browser}{Fl\+\_\+\+Browser}} which lets the user select any set of the lines. The user interface is Macintosh style\+: clicking an item turns off all the others and selects that one, dragging selects all the items the mouse moves over, and ctrl + click (Cmd+click on the Mac OS platform) toggles the items. ~\newline
 Shift + click extends the selection until the clicked item. This is different from how forms did it. ~\newline
 Normally the callback is done when the user releases the mouse, but you can change this with \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{when()}}. 

See \mbox{\hyperlink{class_fl___browser}{Fl\+\_\+\+Browser}} for methods to add and remove lines from the browser.
