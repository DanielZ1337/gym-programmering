\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___slider}{Fl\+\_\+\+Slider}} widget contains a sliding knob inside a box. It is often used as a scrollbar. Moving the box all the way to the top/left sets it to the \mbox{\hyperlink{class_fl___valuator_a408a744457133c6ce66ad3860bc3b3d3}{minimum()}}, and to the bottom/right to the \mbox{\hyperlink{class_fl___valuator_a97b03d366666f9f04bbab8d8340a694b}{maximum()}}. The \mbox{\hyperlink{class_fl___valuator_a408a744457133c6ce66ad3860bc3b3d3}{minimum()}} may be greater than the \mbox{\hyperlink{class_fl___valuator_a97b03d366666f9f04bbab8d8340a694b}{maximum()}} to reverse the slider direction.

Use void Fl\+\_\+\+Widget\+::type(int) to set how the slider is drawn, which can be one of the following\+:

\begin{DoxyItemize}
\item F\+L\+\_\+\+V\+E\+R\+T\+I\+C\+AL -\/ Draws a vertical slider (this is the default). \item F\+L\+\_\+\+H\+O\+R\+I\+Z\+O\+N\+T\+AL -\/ Draws a horizontal slider. \item F\+L\+\_\+\+V\+E\+R\+T\+\_\+\+F\+I\+L\+L\+\_\+\+S\+L\+I\+D\+ER -\/ Draws a filled vertical slider, useful as a progress or value meter. \item F\+L\+\_\+\+H\+O\+R\+\_\+\+F\+I\+L\+L\+\_\+\+S\+L\+I\+D\+ER -\/ Draws a filled horizontal slider, useful as a progress or value meter. \item F\+L\+\_\+\+V\+E\+R\+T\+\_\+\+N\+I\+C\+E\+\_\+\+S\+L\+I\+D\+ER -\/ Draws a vertical slider with a nice looking control knob. \item F\+L\+\_\+\+H\+O\+R\+\_\+\+N\+I\+C\+E\+\_\+\+S\+L\+I\+D\+ER -\/ Draws a horizontal slider with a nice looking control knob.\end{DoxyItemize}
