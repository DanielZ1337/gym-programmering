\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}} class is the F\+L\+TK container widget. It maintains an array of child widgets. These children can themselves be any widget including \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}}. The most important subclass of \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}} is \mbox{\hyperlink{class_fl___window}{Fl\+\_\+\+Window}}, however groups can also be used to control radio buttons or to enforce resize behavior.

The tab and arrow keys are used to move the focus between widgets of this group, and to other groups. The only modifier grabbed is shift (for shift-\/tab), so that ctrl-\/tab, alt-\/up, and such are free for the app to use as shortcuts.

The resizable widget defines the resizing box for the group. When the group is resized it calculates a new size and position for all of its children. Widgets that are horizontally or vertically inside the dimensions of the box are scaled to the new size. Widgets outside the box are moved.

The resizable may be set to the group itself, in which case all the contents are resized. This is the default value for \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}}, although N\+U\+LL is the default for \mbox{\hyperlink{class_fl___window}{Fl\+\_\+\+Window}} and \mbox{\hyperlink{class_fl___pack}{Fl\+\_\+\+Pack}}.

If the resizable is N\+U\+LL then all widgets remain a fixed size and distance from the top-\/left corner.

It is possible to achieve any type of resize behavior by using an invisible \mbox{\hyperlink{class_fl___box}{Fl\+\_\+\+Box}} as the resizable and/or by using a hierarchy of child \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}}\textquotesingle{}s. \mbox{\Hypertarget{class_fl___group_aaec24e2eec53dcea00d1d1f63d5495d3}\label{class_fl___group_aaec24e2eec53dcea00d1d1f63d5495d3}}

{\footnotesize\ttfamily \mbox{\hyperlink{png_8h_aa8c59027f9ab2769342f248709d68d17}{void}} Fl\+\_\+\+Group\+::resize (\begin{DoxyParamCaption}\item[{int}]{x,  }\item[{int}]{y,  }\item[{int}]{w,  }\item[{int}]{h }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [virtual]}}

Changes the size or position of the widget.

This is a virtual function so that the widget may implement its own handling of resizing. The default version does {\itshape not} call the \mbox{\hyperlink{class_fl___widget_aa63ce68cbf4620cf8750b868368ea02b}{redraw()}} method, but instead relies on the parent widget to do so because the parent may know a faster way to update the display, such as scrolling from the old position. ~\newline


Some window managers under X11 call \mbox{\hyperlink{class_fl___group_aaec24e2eec53dcea00d1d1f63d5495d3}{resize()}} a lot more often than needed. Please verify that the position or size of a widget did actually change before doing any extensive calculations.

position(\+X, Y) is a shortcut for resize(X, Y, \mbox{\hyperlink{class_fl___widget_ad59ece778aaf15acc4dd01a8bf5f8012}{w()}}, \mbox{\hyperlink{class_fl___widget_aeeabd61e2d10c76cd6600dbbfe951f71}{h()}}), and size(\+W, H) is a shortcut for resize(\mbox{\hyperlink{class_fl___widget_abea0b9ad493d58bf7fda8d3c81d477b7}{x()}}, \mbox{\hyperlink{class_fl___widget_aec1aded9efdf5f23dfac8cc725cefc4a}{y()}}, W, H).
