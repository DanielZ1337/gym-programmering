\doxysubsection{Detailed Description}
Rich text display widget. 

This is the F\+L\+TK text display widget. It allows the user to view multiple lines of text and supports highlighting, word wrap, mixes of font faces and colors, line numbers and scrolling. The buffer that is displayed in the widget is managed by the \mbox{\hyperlink{class_fl___text___buffer}{Fl\+\_\+\+Text\+\_\+\+Buffer}} class. A single Text Buffer can be displayed by multiple Text Displays.

Changes the size or position of the widget.

This is a virtual function so that the widget may implement its own handling of resizing. The default version does {\itshape not} call the \mbox{\hyperlink{class_fl___widget_aa63ce68cbf4620cf8750b868368ea02b}{redraw()}} method, but instead relies on the parent widget to do so because the parent may know a faster way to update the display, such as scrolling from the old position. ~\newline


Some window managers under X11 call \mbox{\hyperlink{class_fl___text___display_aa8215d8e63bba5f1550ae57e6969263a}{resize()}} a lot more often than needed. Please verify that the position or size of a widget did actually change before doing any extensive calculations.

position(\+X, Y) is a shortcut for resize(X, Y, \mbox{\hyperlink{class_fl___text___display_a0ba3a65c1ff72abb19d00ce743c0444d}{w()}}, \mbox{\hyperlink{class_fl___text___display_abbe11349fa20b8dab805508316d6f1ba}{h()}}), and size(\+W, H) is a shortcut for resize(\mbox{\hyperlink{class_fl___text___display_a553086d9e30bf6745c02ba228e43db1c}{x()}}, \mbox{\hyperlink{class_fl___text___display_a3ec54952411bf446b977c713f4ec46c7}{y()}}, W, H).
