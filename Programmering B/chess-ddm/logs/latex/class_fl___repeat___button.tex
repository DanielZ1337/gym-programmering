\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___repeat___button}{Fl\+\_\+\+Repeat\+\_\+\+Button}} is a subclass of \mbox{\hyperlink{class_fl___button}{Fl\+\_\+\+Button}} that generates a callback when it is pressed and then repeatedly generates callbacks as long as it is held down. The speed of the repeat is fixed and depends on the implementation.
