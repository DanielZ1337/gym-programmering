\doxysubsection{Detailed Description}
A drawing surface that\textquotesingle{}s susceptible to receive graphical output. Any F\+L\+TK application has at any time a current drawing surface to which all drawing requests are directed. The current surface is given by \mbox{\hyperlink{class_fl___surface___device_a6a2643e352fdcb9e78faa167d1cd0a14}{Fl\+\_\+\+Surface\+\_\+\+Device\+::surface()}}. When \mbox{\hyperlink{main_8cpp_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()}} begins running, the current drawing surface has been set to the computer\textquotesingle{}s display, an instance of the \mbox{\hyperlink{class_fl___display___device}{Fl\+\_\+\+Display\+\_\+\+Device}} class.

A drawing surface other than the computer\textquotesingle{}s display, is typically used as follows\+: 
\begin{DoxyEnumerate}
\item Create {\ttfamily surface}, an object from a particular \mbox{\hyperlink{class_fl___surface___device}{Fl\+\_\+\+Surface\+\_\+\+Device}} derived class (e.\+g., \mbox{\hyperlink{class_fl___copy___surface}{Fl\+\_\+\+Copy\+\_\+\+Surface}}, \mbox{\hyperlink{class_fl___printer}{Fl\+\_\+\+Printer}}). 
\item Memorize what is the current drawing surface with {\ttfamily  \mbox{\hyperlink{class_fl___surface___device}{Fl\+\_\+\+Surface\+\_\+\+Device}} $\ast$old\+\_\+current = \mbox{\hyperlink{class_fl___surface___device_a6a2643e352fdcb9e78faa167d1cd0a14}{Fl\+\_\+\+Surface\+\_\+\+Device\+::surface()}};} 
\item Call {\ttfamily surface-\/$>$\mbox{\hyperlink{class_fl___surface___device_a6f88aed1b68854433d60cf23731c3d79}{set\+\_\+current()}}}; to redirect all graphics requests to {\ttfamily surface} which becomes the new current drawing surface (not necessary with class \mbox{\hyperlink{class_fl___printer}{Fl\+\_\+\+Printer}} because it is done by \mbox{\hyperlink{class_fl___printer_a8f3603112736f55dd08ef7ad4aaed551}{Fl\+\_\+\+Printer\+::start\+\_\+job()}}). 
\item At this point any of the \mbox{\hyperlink{group__fl__drawings}{Drawing functions}} (e.\+g., \mbox{\hyperlink{group__fl__drawings_ga690c83e11f49fa837b563f4c0bc4fd1b}{fl\+\_\+rect()}}) or the \mbox{\hyperlink{group__fl__attributes}{Color \& Font functions}} or drawing\+\_\+images functions (e.\+g., \mbox{\hyperlink{group__fl__drawings_gaf9ad52880174bae01461adfa5b4020ff}{fl\+\_\+draw\+\_\+image()}}, \mbox{\hyperlink{class_fl___image_a95e713f20628bfb4c8155ab94ba65ba1}{Fl\+\_\+\+Image\+::draw()}}) operates on the new current drawing surface. Certain drawing surfaces allow additional ways to draw to them (e.\+g., \mbox{\hyperlink{class_fl___printer_a0b6a37315107179fec2eab69008f5150}{Fl\+\_\+\+Printer\+::print\+\_\+widget()}}, \mbox{\hyperlink{class_fl___image___surface_a0c4a97f89c9b0fc091d8d2c0be7da0ce}{Fl\+\_\+\+Image\+\_\+\+Surface\+::draw()}}). 
\item After all drawing requests have been performed, redirect graphics requests back to their previous destination with {\ttfamily old\+\_\+current-\/$>$\mbox{\hyperlink{class_fl___surface___device_a6f88aed1b68854433d60cf23731c3d79}{set\+\_\+current()}}};. 
\item Delete {\ttfamily surface}. 
\end{DoxyEnumerate}
