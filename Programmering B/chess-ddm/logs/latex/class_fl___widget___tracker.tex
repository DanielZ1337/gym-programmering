\doxysubsection{Detailed Description}
This class should be used to control safe widget deletion.

You can use an \mbox{\hyperlink{class_fl___widget___tracker}{Fl\+\_\+\+Widget\+\_\+\+Tracker}} object to watch another widget, if you need to know, if this widget has been deleted during a callback.

This simplifies the use of the \char`\"{}safe widget deletion\char`\"{} methods \mbox{\hyperlink{group__fl__del__widget_gaa6dcacf7538e5424c8366f7a569c0e05}{Fl\+::watch\+\_\+widget\+\_\+pointer()}} and \mbox{\hyperlink{group__fl__del__widget_gabcf12c2424e03d16611f69cf8cf5a033}{Fl\+::release\+\_\+widget\+\_\+pointer()}} and makes their use more reliable, because the destructor autmatically releases the widget pointer from the widget watch list.

It is intended to be used as an automatic (local/stack) variable, such that the automatic destructor is called when the object\textquotesingle{}s scope is left. This ensures that no stale widget pointers are left in the widget watch list (see example below).

You can also create \mbox{\hyperlink{class_fl___widget___tracker}{Fl\+\_\+\+Widget\+\_\+\+Tracker}} objects with {\ttfamily new}, but then it is your responsibility to delete the object (and thus remove the widget pointer from the watch list) when it is not needed any more.
