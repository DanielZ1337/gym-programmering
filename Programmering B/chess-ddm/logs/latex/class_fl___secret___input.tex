\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___secret___input}{Fl\+\_\+\+Secret\+\_\+\+Input}} class is a subclass of \mbox{\hyperlink{class_fl___input}{Fl\+\_\+\+Input}} that displays its input as a string of placeholders. Depending on the platform this placeholder is either the asterisk (\textquotesingle{}$\ast$\textquotesingle{}) or the Unicode bullet character (U+2022).

This subclass is usually used to receive passwords and other \char`\"{}secret\char`\"{} information.
