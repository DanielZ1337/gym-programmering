\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_fl___value___output}{Fl\+\_\+\+Value\+\_\+\+Output}} widget displays a floating point value. If \mbox{\hyperlink{class_fl___valuator_a3308133e6e4963e0e22f4fb405bef298}{step()}} is not zero, the user can adjust the value by dragging the mouse left and right. The left button moves one \mbox{\hyperlink{class_fl___valuator_a3308133e6e4963e0e22f4fb405bef298}{step()}} per pixel, the middle by 10 $\ast$ \mbox{\hyperlink{class_fl___valuator_a3308133e6e4963e0e22f4fb405bef298}{step()}}, and the right button by 100 $\ast$ \mbox{\hyperlink{class_fl___valuator_a3308133e6e4963e0e22f4fb405bef298}{step()}}. 

This is much lighter-\/weight than \mbox{\hyperlink{class_fl___value___input}{Fl\+\_\+\+Value\+\_\+\+Input}} because it contains no text editing code or character buffer.
