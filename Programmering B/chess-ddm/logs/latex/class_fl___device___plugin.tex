\doxysubsection{Detailed Description}
This plugin socket allows the integration of new device drivers for special window or screen types. This class is not intended for use outside the F\+L\+TK library. It is currently used to provide an automated printing service and screen capture for Open\+GL windows, if linked with fltk\+\_\+gl.
