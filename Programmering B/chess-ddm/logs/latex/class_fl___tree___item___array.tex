\doxysubsection{Detailed Description}
Manages an array of \mbox{\hyperlink{class_fl___tree___item}{Fl\+\_\+\+Tree\+\_\+\+Item}} pointers. 

Because F\+L\+TK 1.\+x.\+x. has mandated that templates and S\+TL not be used, we use this class to dynamically manage the arrays.

None of the methods do range checking on index values; the caller must be sure that index values are within the range 0$<$index$<$\mbox{\hyperlink{class_fl___tree___item___array_a1b76ee8f1c8e3d07449bfc6c1a16e9c6}{total()}} (unless otherwise noted).
