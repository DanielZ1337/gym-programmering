A submenu title is identified by the bit F\+L\+\_\+\+S\+U\+B\+M\+E\+NU in the flags field, and ends with a \mbox{\hyperlink{struct_fl___menu___item_aaf2b952945884a7728138e26e6fbc77c}{label()}} that is N\+U\+LL. You can nest menus to any depth. A pointer to the first item in the submenu can be treated as an Fl\+\_\+\+Menu array itself. It is also possible to make separate submenu arrays with F\+L\+\_\+\+S\+U\+B\+M\+E\+N\+U\+\_\+\+P\+O\+I\+N\+T\+ER flags.

You should use the method functions to access structure members and not access them directly to avoid compatibility problems with future releases of F\+L\+TK.
