\doxysubsection{Detailed Description}
This class manages Unicode text displayed in one or more \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}} widgets. 

All text in \mbox{\hyperlink{class_fl___text___buffer}{Fl\+\_\+\+Text\+\_\+\+Buffer}} must be encoded in U\+T\+F-\/8. All indices used in the function calls must be aligned to the start of a U\+T\+F-\/8 sequence. All indices and pointers returned will be aligned. All functions that return a single character will return that in an unsiged int in U\+C\+S-\/4 encoding.

The \mbox{\hyperlink{class_fl___text___buffer}{Fl\+\_\+\+Text\+\_\+\+Buffer}} class is used by the \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}} and \mbox{\hyperlink{class_fl___text___editor}{Fl\+\_\+\+Text\+\_\+\+Editor}} to manage complex text data and is based upon the excellent N\+Edit text editor engine -\/ see \href{http://www.nedit.org/}{\texttt{ http\+://www.\+nedit.\+org/}}.

This function is called after reading a file if the file content was transcoded to U\+T\+F-\/8. Its default implementation calls \mbox{\hyperlink{fl__ask_8_h_a44ce8540fd801b6adaaf5e499c2fc66e}{fl\+\_\+alert()}} with the text of \mbox{\hyperlink{class_fl___text___buffer_af334add7771e7928e7e01fa9a521455a}{file\+\_\+encoding\+\_\+warning\+\_\+message}}. Use \mbox{\hyperlink{class_fl___text___buffer_a3194e63f030767326476d4e719d69a32}{input\+\_\+file\+\_\+was\+\_\+transcoded}} to be informed if file input required transcoding to U\+T\+F-\/8.
