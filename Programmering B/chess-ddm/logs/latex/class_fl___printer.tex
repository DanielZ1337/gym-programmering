\doxysubsection{Detailed Description}
O\+S-\/independent print support. 

\mbox{\hyperlink{class_fl___printer}{Fl\+\_\+\+Printer}} allows to use all drawing, color, text, image, and clip F\+L\+TK functions, and to have them operate on printed page(s). There are two main, non exclusive, ways to use it. 
\begin{DoxyItemize}
\item Print any widget (standard, custom, \mbox{\hyperlink{class_fl___window}{Fl\+\_\+\+Window}}, \mbox{\hyperlink{class_fl___gl___window}{Fl\+\_\+\+Gl\+\_\+\+Window}}) as it appears on screen, with optional translation, scaling and rotation. This is done by calling \mbox{\hyperlink{class_fl___printer_a0b6a37315107179fec2eab69008f5150}{print\+\_\+widget()}}, \mbox{\hyperlink{class_fl___paged___device_a2d230f6507682c810d56ed7b60092dec}{print\+\_\+window()}} or \mbox{\hyperlink{class_fl___printer_afb75fad498e6e1eacd8d66f54a0964fd}{print\+\_\+window\+\_\+part()}}. 
\item Use a series of F\+L\+TK graphics commands (e.\+g., font, text, lines, colors, clip, image) to compose a page appropriately shaped for printing. 
\end{DoxyItemize}In both cases, begin by \mbox{\hyperlink{class_fl___printer_a8f3603112736f55dd08ef7ad4aaed551}{start\+\_\+job()}}, \mbox{\hyperlink{class_fl___printer_aa7c9294391ecebe05a59b868fe16bac1}{start\+\_\+page()}}, \mbox{\hyperlink{class_fl___printer_ae7d6d14dd0c0711634f38c599811db9c}{printable\+\_\+rect()}} and \mbox{\hyperlink{class_fl___printer_ab06af1da74d884b3acc3fd653bfa665e}{origin()}} calls and finish by \mbox{\hyperlink{class_fl___printer_ac3afd751b80f12a11f02a6e87eeb332a}{end\+\_\+page()}} and \mbox{\hyperlink{class_fl___printer_a181c505fbfba8818368f844f7276b06a}{end\+\_\+job()}} calls.

 {\bfseries{Platform specifics}} 
\begin{DoxyItemize}
\item Unix/\+Linux platforms\+: Unless it has been previously changed, the default paper size is A4. To change that, press the \char`\"{}\+Properties\char`\"{} button of the \char`\"{}\+Print\char`\"{} dialog window opened by an \mbox{\hyperlink{class_fl___printer_a8f3603112736f55dd08ef7ad4aaed551}{Fl\+\_\+\+Printer\+::start\+\_\+job()}} call. This opens a \char`\"{}\+Printer Properties\char`\"{} window where it\textquotesingle{}s possible to select the adequate paper size. Finally press the \char`\"{}\+Save\char`\"{} button therein to assign the chosen paper size to the chosen printer for this and all further print operations. ~\newline
Class \mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} prints but loses its transparency if it has one. See class \mbox{\hyperlink{class_fl___post_script___graphics___driver}{Fl\+\_\+\+Post\+Script\+\_\+\+Graphics\+\_\+\+Driver}} for a description of how U\+T\+F-\/8 strings appear in print. Use the static public attributes of this class to set the print dialog to other languages than English. For example, the \char`\"{}\+Printer\+:\char`\"{} dialog item \mbox{\hyperlink{class_fl___printer_addb8c90d951d73581750bb3249129b4b}{Fl\+\_\+\+Printer\+::dialog\+\_\+printer}} can be set to French with\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_fl___printer_addb8c90d951d73581750bb3249129b4b}{Fl\_Printer::dialog\_printer}} = \textcolor{stringliteral}{"Imprimante:"};}
\end{DoxyCode}
 before creation of the \mbox{\hyperlink{class_fl___printer}{Fl\+\_\+\+Printer}} object. Use \mbox{\hyperlink{class_fl___post_script___file___device_a3de1d5150d6da464b821bc18e798deb2}{Fl\+\_\+\+Post\+Script\+\_\+\+File\+\_\+\+Device\+::file\+\_\+chooser\+\_\+title}} to customize the title of the file chooser dialog that opens when using the \char`\"{}\+Print To File\char`\"{} option of the print dialog. 
\item M\+S\+Windows platform\+: Transparent \mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} \textquotesingle{}s don\textquotesingle{}t print with exact transparency on most printers. \mbox{\hyperlink{class_fl___r_g_b___image}{Fl\+\_\+\+R\+G\+B\+\_\+\+Image}} \textquotesingle{}s don\textquotesingle{}t \mbox{\hyperlink{class_fl___printer_a2c5e858532b376ee7627369c9c8b1c36}{rotate()}} well. A workaround is to use the \mbox{\hyperlink{class_fl___printer_afb75fad498e6e1eacd8d66f54a0964fd}{print\+\_\+window\+\_\+part()}} call. 
\item Mac OS X platform\+: all graphics requests print as on display. 
\end{DoxyItemize}
