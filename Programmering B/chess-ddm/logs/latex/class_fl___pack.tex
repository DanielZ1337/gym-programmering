\doxysubsection{Detailed Description}
This widget was designed to add the functionality of compressing and aligning widgets. 

If \mbox{\hyperlink{class_fl___widget_adda2e8f162b3ea0332eb8d762350bbd5}{type()}} is \mbox{\hyperlink{class_fl___pack_ae78e492675cdb1e7c8c4163889f165dea6abdf21990ade67cabbdf1d34f6b9b31}{Fl\+\_\+\+Pack\+::\+H\+O\+R\+I\+Z\+O\+N\+T\+AL}} all the children are resized to the height of the \mbox{\hyperlink{class_fl___pack}{Fl\+\_\+\+Pack}}, and are moved next to each other horizontally. If \mbox{\hyperlink{class_fl___widget_adda2e8f162b3ea0332eb8d762350bbd5}{type()}} is not \mbox{\hyperlink{class_fl___pack_ae78e492675cdb1e7c8c4163889f165dea6abdf21990ade67cabbdf1d34f6b9b31}{Fl\+\_\+\+Pack\+::\+H\+O\+R\+I\+Z\+O\+N\+T\+AL}} then the children are resized to the width and are stacked below each other. Then the \mbox{\hyperlink{class_fl___pack}{Fl\+\_\+\+Pack}} resizes itself to surround the child widgets. 

This widget is needed for the \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}}. In addition you may want to put the \mbox{\hyperlink{class_fl___pack}{Fl\+\_\+\+Pack}} inside an \mbox{\hyperlink{class_fl___scroll}{Fl\+\_\+\+Scroll}}.

The resizable for \mbox{\hyperlink{class_fl___pack}{Fl\+\_\+\+Pack}} is set to N\+U\+LL by default.

See also\+: \mbox{\hyperlink{class_fl___group_ab0ed0d1974d7185594687849f386c5f3}{Fl\+\_\+\+Group\+::resizable()}}
