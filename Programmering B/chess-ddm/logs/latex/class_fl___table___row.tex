\doxysubsection{Detailed Description}
A table with row selection capabilities.

This class implements a simple table with the ability to select rows. This widget is similar to an \mbox{\hyperlink{class_fl___browser}{Fl\+\_\+\+Browser}} with columns. Most methods of importance will be found in the \mbox{\hyperlink{class_fl___table}{Fl\+\_\+\+Table}} widget, such as \mbox{\hyperlink{class_fl___table_af63dddf97c8804488f8a179780a674c0}{Fl\+\_\+\+Table\+::rows()}} and \mbox{\hyperlink{class_fl___table_a82b1a353a6d0345e6714fb7d3a2ae0ac}{Fl\+\_\+\+Table\+::cols()}}.

To be useful it must be subclassed and at minimum the \mbox{\hyperlink{class_fl___table_a3f1f1ccc017d9159c1c98f6f7268bf6d}{draw\+\_\+cell()}} method must be overridden to provide the content of the cells. This widget does {\itshape not} manage the cell\textquotesingle{}s data content; it is up to the parent class\textquotesingle{}s \mbox{\hyperlink{class_fl___table_a3f1f1ccc017d9159c1c98f6f7268bf6d}{draw\+\_\+cell()}} method override to provide this.

Events on the cells and/or headings generate callbacks when they are clicked by the user. You control when events are generated based on the values you supply for \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{Fl\+\_\+\+Table\+::when()}}.
