\doxysubsection{Detailed Description}
This widget is based off the \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}} widget, but instead of displaying tabs it only changes \char`\"{}tabs\char`\"{} under program control. Its primary purpose is to support \char`\"{}wizards\char`\"{} that step a user through configuration or troubleshooting tasks.

As with \mbox{\hyperlink{class_fl___tabs}{Fl\+\_\+\+Tabs}}, wizard panes are composed of child (usually \mbox{\hyperlink{class_fl___group}{Fl\+\_\+\+Group}}) widgets. Navigation buttons must be added separately.
