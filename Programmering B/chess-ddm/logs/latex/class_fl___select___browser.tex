\doxysubsection{Detailed Description}
The class is a subclass of \mbox{\hyperlink{class_fl___browser}{Fl\+\_\+\+Browser}} which lets the user select a single item, or no items by clicking on the empty space. As long as the mouse button is held down on an unselected item it is highlighted. Normally the callback is done when the user presses the mouse, but you can change this with \mbox{\hyperlink{class_fl___widget_ad4ef380ffb50f4d00fe93f7b6677e9e1}{when()}}. 

See \mbox{\hyperlink{class_fl___browser}{Fl\+\_\+\+Browser}} for methods to add and remove lines from the browser.
