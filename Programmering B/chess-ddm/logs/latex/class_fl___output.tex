\doxysubsection{Detailed Description}
This widget displays a piece of text.

When you set the \mbox{\hyperlink{class_fl___input___aaccaca04083dc6d2f1c1a4fa273d9048}{value()}} , \mbox{\hyperlink{class_fl___output}{Fl\+\_\+\+Output}} does a strcpy() to its own storage, which is useful for program-\/generated values. The user may select portions of the text using the mouse and paste the contents into other fields or programs.



 
\begin{DoxyImage}
\includegraphics[width=8cm]{text.png}
\doxyfigcaption{Fl\+\_\+\+Output}
\end{DoxyImage}


There is a single subclass, \mbox{\hyperlink{class_fl___multiline___output}{Fl\+\_\+\+Multiline\+\_\+\+Output}}, which allows you to display multiple lines of text. \mbox{\hyperlink{class_fl___multiline___output}{Fl\+\_\+\+Multiline\+\_\+\+Output}} does not provide scroll bars. If a more complete text editing widget is needed, use \mbox{\hyperlink{class_fl___text___display}{Fl\+\_\+\+Text\+\_\+\+Display}} instead.

The text may contain any characters except \textbackslash{}0, and will correctly display anything, using $^\wedge$X notation for unprintable control characters and \textbackslash{}nnn notation for unprintable characters with the high bit set. It assumes the font can draw any characters in the I\+S\+O-\/\+Latin1 character set.
