Detailed Description

All graphical output devices and all graphics systems. This class supports a rudimentary system of run-time type information.

virtual const char* Fl_Device::class_name ( )

Returns the name of the class of this object. Use of the class_name() function is discouraged because it will be removed from future FLTK versions.

The class of an instance of an Fl_Device subclass can be checked with code such as:

if ( instance->class_name() == Fl_Printer::class_id ) { ... }
